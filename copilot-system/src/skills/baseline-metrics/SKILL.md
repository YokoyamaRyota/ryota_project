# Baseline Metrics Skill

## Purpose

- KPI-1 から KPI-5 の初期値を確定する。

## Steps

1. 複雑度ごとに代表タスクを実行。
2. lead_time, cost, intent_compliance, critical_miss, decision_sla を計測。
3. 形式を揃えて baseline 文書へ記録。
4. lockdate 到達後は基準値を固定し比較対象にする。
